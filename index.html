Yêu cầu khi viết JavaScript cần clean

  • **Hiểu rõ bài toán**: Trước khi viết code, hãy đảm bảo em hiểu rõ yêu cầu và mục tiêu của bài toán.
  • **Chia nhỏ bài toán**: Phân chia bài toán thành các chức năng nhỏ, dễ quản lý như thêm, xóa, cập nhật công việc.
  • **Viết code dễ đọc**: Sử dụng tên biến và hàm rõ ràng, mô tả đúng chức năng. Ví dụ: `addTodo`, `deleteTodo`.
  • **Tránh lặp code**: Nếu một đoạn code được sử dụng nhiều lần, hãy tách nó thành một hàm riêng.
  • **Kiểm tra dữ liệu đầu vào**: Đảm bảo dữ liệu đầu vào hợp lệ trước khi xử lý, ví dụ: không cho phép thêm công việc trống.
  • **Sử dụng AI một cách có trách nhiệm**: AI có thể hỗ trợ viết code, nhưng em cần hiểu rõ code được tạo ra để đảm bảo nó phù hợp với yêu cầu.
  • **Kiểm tra và tối ưu hóa**: Sau khi hoàn thành, kiểm tra toàn bộ ứng dụng và tối ưu hóa code nếu cần.
